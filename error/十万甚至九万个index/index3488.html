恶俗维基:隱私政策

  • 以其他語言閱讀
  • 監視此頁面
  • 編輯

為了使訪客或註冊用戶（以下簡稱「用戶」）能夠更好的使用惡俗維基（以下簡稱「本站」），本站可能需要收集一些個人信息，本隱私策略用於讓用戶知曉本站收集哪些信息，以及本站如何處理用戶的個人信息。

目錄

GDPR 豁免

根據歐盟數據保護通用條例（GDPR）的規定，本站的運營主體為個人，因此本站無需遵守 GDPR，也無需發布數據處理單位的身份。

數據處理單位

名稱：無

地址：無

聯繫方式：本站首頁發布的聯繫方式

收集信息的方式

  1. 用戶向本站提交的內容，例如，用戶在編輯框中輸入了內容，並點擊提交按鈕
  2. 網站的自動化設施，例如 Javascript 和伺服器日誌記錄

使用信息的方式

  1. 本站使用伺服器日誌記錄，以確保本站的信息安全
  2. 本站使用部分個人信息，以確保本站的內容免遭破壞
  3. 本站使用 Google Analytics 生成的統計信息，以分析網站訪客來源等
  4. 在評論區匿名發表評論，並提及他人或使用回復功能時，被提及/回復的人將得到您的 IP 地址
  5. 在評論區違反評論規則時，您的 IP 地址將公開出現在封禁列表中
  6. 部分情況下，本站管理員可能會使用收集到的信息，但不會分享該信息

分享信息的方式

除本頁面所述之外，本站不會分享收集到的相關信息。

第三方內容

  1. 本站使用了 Google Analytics 以收集統計信息，關於 Google Analytics，請參閱 Google 隱私政策
  2. 本站使用了 Google reCAPTCHA 以保護網站免受自動化程序的破壞，關於 Google reCAPTCHA，請參閱 Google 隱私政策
  3. 本站使用了 CloudFlare 作為內容分發網絡，關於 CloudFlare，請參閱 CloudFlare 隱私政策
  4. 本站可能包含外部連結，本站不對外部連結的內容、安全性或信息收集等行為負責

用戶權利

  1. 用戶有權通過複製等操作獲取用戶發布的信息副本，根據版權策略，用戶發布內容代表向本站永久授權，因此，本站可能不會刪除用戶發布的信息
  2. 用戶有權刪除其發布的內容，但是如果要刪除內容，應當經過所有貢獻過該頁面的用戶同意
  3. 用戶有權請求刪除其他用戶發布的內容，但應當經過所有貢獻過該頁面的用戶同意
  4. 用戶有權刪除其帳號
  5. 其他法律賦予用戶的權利

數據的刪除和變更

  1. 當本站上的內容被刪除或修改後，本站保留還原該內容的權利，本站不保證也無法保證其他用戶或第三方網站將原內容進行存檔
  2. 本站無條件執行真實的法院判決結果，但本站無法做到的除外。此外，為了保護用戶信息，對於獲取用戶個人資料的請求，本站只處理來自法院判決的該類請求
  3. 本站可能會執行來自執法機構的命令，除非明確要求保密，否則無論本站是否執行該命令，都會將該請求的內容公開
  4. 本站會按實際情況酌情處理來自用戶的請求，例如刪除內容，修改用戶信息，糾正不實信息等。請注意，刪除內容請求只有當符合下列條件之一時才會執行：
    1. 大部分用戶同意刪除該內容
    2. 所有編輯過該頁面的人同意刪除該內容
    3. 違反美國法律的內容
    4. 違反惡俗維基守則的內容
    5. 其他特殊情況

數據保護

  1. 本站採用 TLS 加密技術以確保數據傳輸的安全
  2. 本站採用 Web 應用防火牆（WAF）以降低網站受到黑客攻擊的可能性
  3. 本站嚴格遵循金融行業數據安全標準（PCI DSS）規範並已進行評估：
    1. 本站使用多重安全服務和防火牆
    2. 本站使用了加密技術進行數據存儲和傳輸
    3. 本站為所有系統執行定期檢查，並確保作業系統或軟體不存在已知漏洞
    4. 本站嚴格限制伺服器的管理權限，並擁有安全審計日誌
  4. 本站無法保證也不可能保證數據的絕對安全

技術細節或詳細信息

  1. 本站採用 Cookie 技術以分析用戶的訪問情況，或保存用戶的登錄會話，關於 Cookie 以及 Cookie 如何工作，請參閱 Cookie
  2. 本站的伺服器會自動記錄日誌文件，日誌文件包含以下信息
    1. 用戶的 IP 地址
    2. 反向代理伺服器的 IP 地址
    3. URL（網址）
    4. 來源 URL
    5. 時間
    6. 用戶代理字符串（User-Agent）
  3. 當用戶向本站主動提交內容時，例如在輸入框中輸入內容並點擊提交按鈕，伺服器可能會記錄任何用戶主動提交的信息
